Предпосылки создания развитых редакторов кода

Как известно, стандарт HTML представляет документ HTML в виде обычного текстового файла. Поэтому для простого редактирования HTML файлов вполне хватает небольшого текстового редактора, желательно с подсветкой синтаксиса, например, хорошо известного Notepad++. Но современный стандарт HTML5, как и HTML4, вполне допускает включение в документ фрагментов кода, написанных на другом языке, например, CSS или JavaScript. Документы PHP также могут быть представлены как обычные файлы HTML со вставками кода на языке PHP. К сожалению, в таких документах с переменным синтаксисом Notepad++ не может автоматически переключать подсветку кода.

Кроме того, при создании большого проекта со сложной структурой, с большим количеством ссылок, весьма желательно иметь развитую систему навигации по всему проекту. Например, давать возможность быстрого вызова диалога выбора файла на локальном компьютере в том месте документа, где по смыслу ожидается ссылка на файл. Ответом на эти потребности Web дизайнеров стало появление развитых редакторов кода, предназначенных для редактирования не только одного определенного вида документа, а скорее всего набора документов включенных в проект.

Поэтому в качестве помощи начинающим Web дизайнерам я решил создать подборку информации о существующем программном обеспечении для создания и редактирования Web страниц. Также я решил включить информацию о книгах и ресурсах, которые могут быть полезны для образования и/или повышения квалификации Web дизайнеров. Представленная информация структурирована по используемым языкам программирования или разметки, хотя такое деление может быть произведено весьма условно по причинам, описанным выше. Конечно, в Интернете существует масса подобных страниц, однако я не видел, чтобы такая информация (программы, книги, ресурсы) была вся собрана в одном месте.
